# 📊 Telecom Customer Churn Analysis (EDA)

## 🎯 Project Overview
This project performs an **Exploratory Data Analysis (EDA)** on a Telecommunications dataset to identify the key factors that lead to customer churn. The goal is to provide actionable insights to help the company improve customer retention.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Tools:** Jupyter Notebook

## 🔍 Key Insights from the Analysis
* **Tenure Impact:** Customers with shorter tenure (less than 6 months) are more likely to churn.
* **Contract Type:** Monthly contract customers show a higher churn rate compared to two-year contract holders.
* **Monthly Charges:** Higher monthly charges correlate with increased churn probability.
* **Technical Support:** Customers without tech support services tend to leave more often.

## 📈 Visualizations
The analysis includes:
* **Churn Distribution:** Understanding the balance of the dataset.
* **Correlation Heatmaps:** Identifying relationships between numeric features.
* **Boxplots & Histograms:** Analyzing tenure and monthly charges distributions.

## 🚀 How to Run
1. Clone this repository.
2. Install dependencies: `pip install pandas matplotlib seaborn`.
3. Open `Telecom_Churn_EDA.ipynb` in Jupyter Notebook.
